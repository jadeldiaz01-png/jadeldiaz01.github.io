Propósito de la aplicación

Reels Meta Production ayuda a usuarios autorizados a preparar, revisar, aprobar y publicar videos propios en plataformas como YouTube. La aplicación utiliza las API oficiales de las plataformas y los mecanismos OAuth 2.0 correspondientes.

Cuando un usuario conecta su cuenta de Google o YouTube, la aplicación solicita únicamente los permisos necesarios para ejecutar las funciones seleccionadas, como subir videos al canal de YouTube expresamente autorizado por el usuario.

Funcionamiento general

  1. El usuario conecta voluntariamente una cuenta autorizada.
  2. El usuario proporciona o selecciona el contenido que desea publicar.
  3. El sistema valida el archivo y sus metadatos.
  4. La publicación requiere una decisión y aprobación autorizada.
  5. La API oficial de la plataforma ejecuta la publicación.

Uso de YouTube API Services

Reels Meta Production utiliza YouTube API Services para permitir que un usuario autorizado publique contenido en su propio canal de YouTube.

La aplicación no solicita la contraseña de Google o YouTube. La autorización se realiza directamente mediante el sistema OAuth 2.0 de Google.

Control del usuario

Los usuarios conservan el control de sus cuentas y pueden revocar el acceso concedido a la aplicación desde la configuración de seguridad de su cuenta de Google.
