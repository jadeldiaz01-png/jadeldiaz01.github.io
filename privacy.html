Política de Privacidad de Reels Meta Production

Fecha de entrada en vigor: 10 de julio de 2026

1. Identidad y propósito

Reels Meta Production es una aplicación de automatización supervisada que permite a usuarios autorizados preparar, revisar y publicar contenido propio en plataformas de redes sociales.

Esta política explica qué información puede procesar la aplicación, cómo se utiliza, cómo se protege y qué opciones tienen los usuarios.

2. Uso de YouTube API Services

Reels Meta Production utiliza YouTube API Services para proporcionar funciones relacionadas con la publicación de videos en canales de YouTube expresamente autorizados por sus propietarios o administradores.

El uso de YouTube mediante esta aplicación también está sujeto a las Condiciones del Servicio de YouTube:

Condiciones del Servicio de YouTube

Puedes consultar la Política de Privacidad de Google en:

Política de Privacidad de Google

3. Información a la que puede acceder la aplicación

Dependiendo de las funciones autorizadas por el usuario, la aplicación puede procesar:

  • Tokens de autorización OAuth 2.0 emitidos por Google.
  • Identificadores técnicos de la cuenta o canal autorizado.
  • Videos, títulos, descripciones, etiquetas y configuraciones de privacidad proporcionados por el usuario para su publicación.
  • Resultados técnicos de las operaciones de carga y publicación.
  • Registros de seguridad, errores, auditoría y funcionamiento.

Reels Meta Production no solicita ni almacena la contraseña de la cuenta de Google o YouTube.

4. Cómo se utiliza la información

La información se utiliza exclusivamente para:

  • Autenticar al usuario mediante OAuth 2.0.
  • Ejecutar las funciones autorizadas por el usuario.
  • Subir contenido al canal seleccionado.
  • Validar el estado de una publicación.
  • Prevenir errores, accesos no autorizados y abuso.
  • Conservar registros técnicos y de auditoría necesarios.

5. Compartición de información

La aplicación puede transmitir la información necesaria a Google o YouTube para ejecutar las operaciones expresamente solicitadas por el usuario.

También pueden utilizarse proveedores técnicos de infraestructura, almacenamiento, bases de datos o monitoreo, sujetos a medidas razonables de seguridad y únicamente para prestar el servicio.

Reels Meta Production no vende datos personales ni datos obtenidos mediante Google o YouTube API Services.

6. Almacenamiento y seguridad

Los tokens y demás credenciales deben almacenarse mediante controles de acceso, cifrado o servicios especializados de gestión de secretos.

Se aplican medidas técnicas y organizativas razonables para evitar pérdida, exposición, modificación o acceso no autorizado.

7. Conservación y eliminación

Los datos se conservan solamente durante el tiempo necesario para proporcionar las funciones autorizadas, mantener la seguridad, cumplir obligaciones legales o resolver incidentes.

El usuario puede solicitar la eliminación de sus datos y credenciales enviando un mensaje a:

jadeldiaz01@gmail.com

8. Revocación del acceso de Google

Los usuarios pueden revocar el acceso de Reels Meta Production desde la sección de conexiones y seguridad de su cuenta de Google:

Administrar conexiones con aplicaciones de terceros

La revocación impide que la aplicación genere nuevos accesos mediante la autorización revocada.

9. Uso limitado de datos de Google

El uso y transferencia de información recibida mediante las API de Google se ajustará a la Política de Datos de Usuario de los Servicios de API de Google, incluidos sus requisitos de uso limitado.

10. Privacidad de menores

La aplicación no está dirigida intencionalmente a menores que no tengan la edad legal necesaria para administrar una cuenta o conceder una autorización vinculante.

11. Cambios en esta política

Esta política puede actualizarse cuando cambien las funciones, los requisitos legales o las políticas de las plataformas. La fecha de entrada en vigor se actualizará en esta página.
